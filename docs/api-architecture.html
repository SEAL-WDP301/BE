seal-wdp301-backend

Kiến trúc API

Hệ thống quản lý hackathon SEAL — monolith NestJS với routing theo vai trò (student, mentor, judge, organizer). Mọi endpoint dùng prefix /api, tài liệu Swagger tại /api/docs.

NestJS 10 TypeScript Prisma + PostgreSQL Redis JWT + OAuth2 Socket.IO

Tổng quan

Đây là một ứng dụng monolithic — tất cả module nằm trong một process Node.js, tổ chức theo feature modules dưới src/modules/. Cross-cutting concerns (guards, filters, interceptors) nằm ở src/common/.

Vòng đời Request

Mỗi HTTP request đi qua pipeline sau (định nghĩa trong src/bootstrap.ts):

1
Middleware — bootstrap.ts

Helmet (security headers) → CORS → cookie-parser → RequestLoggerMiddleware (gán X-Request-Id).

2
Guards — JwtAuthGuard + RolesGuard

JWT Bearer token được verify bởi JwtStrategy, load user vào req.user. RolesGuard kiểm tra @Roles() decorator.

3
Pipe + Handler — Controller → Service

ValidationPipe validate DTO → Controller gọi Service → Prisma ghi/đọc PostgreSQL. TransformInterceptor bọc response: { success, message, data, timestamp }.

4
Error — AllExceptionsFilter

Lỗi validation/auth trả về envelope thống nhất: { success: false, statusCode, message }.

Ví dụ: Judge chấm điểm submission — PUT /api/judge/submissions/:id/scores → JWT verify → role stakeholder → SubmissionJudgeService → ghi bảng Score qua Prisma.

Authentication

Cơ chếFlow
Email/Password POST /auth/signup → OTP 6 số (Redis, 5 phút) → verify-otp → kích hoạt tài khoản
JWT Access token (Bearer header) + Refresh token (body/cookie, hash trong DB)
Google OAuth GET /auth/google → callback → redirect frontend kèm tokens
GitHub OAuth GET /auth/github → callback → redirect frontend kèm tokens
WebSocket JWT từ Authorization header hoặc auth.token trong handshake

Dev Quickstart

npm install
cp .env.example .env        # cấu hình DATABASE_URL, REDIS, JWT...
npm run prisma:generate
npm run prisma:migrate
npm run start:dev           # http://localhost:3000/api

# Swagger UI
open http://localhost:3000/api/docs

# Redis local (Docker)
docker compose up -d
